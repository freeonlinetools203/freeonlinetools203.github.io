📐 3. Color Formats Explained

Our color picker randomizer displays colors in four different formats. Understanding these formats is essential for using colors effectively in different contexts.

#️⃣ HEX Color Code

HEX codes are 6-digit alphanumeric codes representing colors. Example: #FF5733

🔴 RGB Color Code

RGB values represent red, green, and blue components. Example: rgb(255, 87, 51)

🌈 HSL Color Code

HSL represents hue, saturation, and lightness. Example: hsl(12, 100%, 60%)

🖨️ CMYK Color Code

CMYK is used for printing. Example: cmyk(0%, 66%, 80%, 0%)
